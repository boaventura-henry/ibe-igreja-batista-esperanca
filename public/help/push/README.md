# Capturas reais do assistente de notificacoes

Use esta pasta somente para capturas reais do assistente de notificacoes.

- Use nomes por sistema e etapa, como `windows-permissao-passo-1.png`.
- Oculte CPF, telefone, e-mail, nomes, IDs, tokens e demais dados pessoais.
- Nao adicione imagens ficticias ou capturas com informacoes do Neon, Vercel ou GitHub.
- O assistente deve funcionar sem depender de imagens.
