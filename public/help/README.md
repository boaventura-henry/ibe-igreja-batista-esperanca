# Imagens da Central de Ajuda

Diretorio reservado para capturas reais do sistema. A Central de Ajuda funciona sem imagens e nenhuma captura ficticia deve ser adicionada.

## Como adicionar uma captura

- Use somente capturas reais feitas em ambiente autorizado.
- Remova ou cubra CPF, telefone, e-mail, nomes, IDs, tokens e qualquer outro dado pessoal antes de adicionar o arquivo.
- Prefira dados de teste e nunca capture credenciais ou informacoes do Neon, Vercel ou GitHub.
- Use PNG ou WebP, com nomes previsiveis no formato `modulo-tela-descricao.png`.
- Atualize o artigo correspondente em `src/content/help/help-content.ts` para referenciar a imagem somente quando ela estiver realmente disponivel.
